# Vanity - AI Discord Bot

## Overview
Vanity is an AI-powered Discord bot that has undergone 9 major rewrites since December 2017. Currently on the "Kale" update (March 2026), Vanity has evolved from a simple chatbot to a full AI agent with voice recognition, voice output, and Minecraft automation capabilities.

## Update History

### Kale Update (March 2026) - Biggest Update Yet
Vanity transformed from chatbot to full AI agent:
- **Voice Recognition** - Listen and respond to voice commands
- **Voice Output** - Spoken responses for hands-free interaction
- **Agent Architecture** - Uses agents and tools for complex tasks
- **Minecraft Automation** - Plays autonomously on server as assistant
- **Note Taking** - Can take notes and edit files on behalf of users
- **File Editing** - Read, write, and modify files
- **Online Research** - Web search and information gathering

### Jicama Update (January 2026)
Focus on Minecraft gameplay as foundation for AI development:
- Combat mechanics and survival strategies
- Item collection and resource gathering
- Tool understanding and usage
- Inventory management
- Sorting and organizing chests
- Fishing and prerequisites

### Ivy Update (May 2025)
Explored real-time gaming AI VTuber potential:
- Modular core architecture for seamless updates
- Ultra-fast voice with reduced latency
- Smarter conversation parsing algorithm
- VTuber integration with fully-rigged avatar
- Controller support for dynamic movement
- Twitch connectivity with read/reply system
- DeepSeek-v3 for response generation
- Experimental VB-Cable audio routing

### Honeynut Update (November 2024)
Speed and stability improvements:
- New AI backend: Llama 3.1 70B via Groq API
- Enhanced voice conversations with new parsing
- Removed old GPT4All and Ollama backends
- Focus on voice stability and controller responsiveness

### Garlic Update (September 2024)
- Model: Dolphin-2.5-mixtral-8x7b
- Switched from GPT4All to Ollama for better performance
- Removed cogs and simplification functions
- Fixed slash commands and AI response disconnect issues
- Removed voice module output to Discord (too slow)

### Fennel Update (June 2024)
- Model: nous-hermes-2-solar-10.7b.Q5_K_M
- Hardware upgrade to 64GB RAM
- Role-based responses
- OpenVoice for speech synthesis

### Eggplant Update (January 2024) - Fifth Iteration
Complete overhaul:
- Switched from JavaScript to Python
- GPT4All with Nous-Hermes-2-solar-10.7b model
- OpenVoice voice responses
- Multiple personalities based on user roles
- Removed Minecraft playing feature (temporarily)

### Diakon Update (November 2023) - Fourth Iteration
- Enhanced in-game interaction
- Improved Minecraft playing (alpha)
- Forge server connectivity attempts

### Carrot Update (August 2023) - Third Iteration
- Enhanced chat with arrays and dictionaries
- Minecraft integration via Mineflayer.js (alpha)

### Broccoli Update (May 2018) - Second Iteration
- Regex-based message recognition
- Announcement repeater
- Direct messaging capability

### Asparagus Update (December 2017) - First Iteration
- JavaScript with Node.js and Discord.js
- Basic commands like chat pruning
- Tutorial-based foundation

## Technical Evolution
- **Languages**: JavaScript → Python
- **AI Models**: Various → GPT4All → Ollama → Groq API (Llama 3.1 70B)
- **Voice**: None → OpenVoice → Ultra-fast voice
- **Architecture**: Simple bot → Modular → Agent-based
