# Coffee Block Minecraft Server

## Overview
Coffee Block is osyra42's Minecraft server community, running for 14+ years (as of 2026). Free to play, no whitelist, no application required. Java Edition only.

## Server Philosophy
- **Free Access**: No donations required, no pay-to-win, no premium ranks
- **Community-First**: Rules and changes require majority approval from active players
- **Real Hardware**: Running on actual hardware (6700K CPU, 32GB RAM, 512GB SSD), not cheap cloud VMs
- **Admin Presence**: osyra42 actively plays and administers

## Server Rules

1. **Self Care**: Prioritize real life - eat, sleep, handle responsibilities. Game second.

2. **Server Damage**: No intentional lag/crash attempts. No lag machines or destructive chunk loaders. Instant ban if intentional.

3. **No Disrespect**: Keep civil. Trash talk during PvP okay, personal attacks/toxicity not allowed.

4. **Player Consent**: Must ask before griefing, fighting, raiding, or looting another player. If player doesn't consent, must respect. No exceptions.

5. **Conversations**: Keep chat clean. No sex, violence, politics, or hate speech.

6. **Be Independent**: Build your own stuff. Teaming requires mutual agreement. No moving into someone's base uninvited.

7. **Hacks Allowed**: Client mods fine (X-ray, minimaps, sorting, shaders). Only rule: don't damage server or ruin others' experience without consent.

8. **Vote Ban**: Community can vote to ban problematic players. Majority rules.

9. **Changes**: Server/rule changes need majority approval from active players. No unilateral decisions.
