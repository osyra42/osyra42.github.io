Clutter ~ Chrome Extension

Clutter is a Chrome extension that adds festive seasonal decorations falling on every webpage you visit! It automatically detects the current season and displays theme-appropriate decorationsâ€”from winter snowflakes to autumn leaves. The decorations feature interactive physics and bounce away from your mouse cursor.

Seasonal Themes: Winter, Spring, Summer, Fall
Holiday Themes: Christmas, Easter, Halloween, Thanksgiving, Valentine's Day, St. Patrick's Day, July 4th, New Year's, Leap Day

Features 54+ unique decoration images with dynamic date calculation for Easter using the Gregorian algorithm.

Installation

1. Download or clone the repository from GitHub
2. Open Chrome and navigate to chrome://extensions/
3. Enable "Developer mode" in the top right corner
4. Click "Load unpacked" and select the clutter folder
5. The extension will now be active on all webpages
