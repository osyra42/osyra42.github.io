What is ComfyUI?

ComfyUI is a powerful, node-based interface for Stable Diffusion and other generative AI models. Unlike traditional UIs with simple text boxes, ComfyUI lets you build complex image generation workflows by connecting nodes together visually - similar to Blender's shader nodes or Unreal's Blueprints.

Why use ComfyUI?

  • Visual workflow - See exactly how your generation pipeline works
  • Reusable workflows - Save and share complex setups as JSON files
  • Fine control - Adjust every parameter in the generation process
  • Efficient - Only re-runs changed parts of your workflow
  • Extensible - Huge library of custom nodes from the community

For installation and getting started, see the official ComfyUI documentation.

Negative Prompt

A good negative prompt helps avoid common generation artifacts. Here's the one I use:

fused mouth, blurry mouth, ugly mouth, fused cheek, blurry cheek, ugly cheek, too many fingers, bad feet, deformed, bad body, malformed limbs, extra limb, extra arms, extra legs, acnes, long body, ugly, bad hands, lowres, bad anatomy, text, error, missing fingers, extra digit, fewer digits, cropped, worst quality, low quality, normal quality, jpeg artifacts, signature, watermark, username, blurry, (3d:1.3), broken limb, incorrect limb, fusion finger, lost finger, multiple fingers, multiple digits, fusion hand, lost leg, fused leg, multiple legs, jpeg, forehead marks, forehead jewels, censored

Key things this addresses:

  • Hands/fingers - Common problem area for AI generation
  • Body proportions - Prevents extra limbs and deformations
  • Quality tags - Filters out low-quality outputs
  • Artifacts - Removes watermarks, signatures, and compression artifacts
  • Style control - (3d:1.3) reduces unwanted 3D render look

Example: Osyra Character

Here's an example prompt for generating my character Osyra:

score_9, score_8_up, score_7_up, best quality, 4k, masterpiece, digital art,

1girl, solo, (brown skin:1.7), (black hair:1.8), long hair, wolf ears, wolf tail, red eyes, adult woman, round glasses, swept bangs,

brown jacket unzipped, jacket with a hood, light yellow shirt, and blue pants,

Breaking down the prompt:

  • Quality tags - score_9, score_8_up, score_7_up are used by Pony-based models to filter for high-quality outputs. Additional tags like "best quality, 4k, masterpiece" reinforce this.
  • Character basics - 1girl, solo establishes a single female character. The weighted tags (brown skin:1.7) and (black hair:1.8) use higher emphasis to ensure these features appear consistently.
  • Physical features - wolf ears, wolf tail, red eyes, round glasses, swept bangs define the character's distinct look.
  • Outfit - Clothing is described last: brown hooded jacket (unzipped), light yellow shirt underneath, and blue pants.

Prompt structure tips:

  • Weight syntax - Use (tag:1.5) to increase emphasis, higher = stronger
  • Order matters - Put important features early in the prompt
  • Commas - Separate distinct concepts with commas
  • Line breaks - Group related concepts together for readability
