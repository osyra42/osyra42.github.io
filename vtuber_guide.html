1. Standard Method (3D Avatar)

1.1 REFERENCE IMAGES
    Before designing, gather visual references for colors, proportions, and style.

    Sources:
    ├─ Pixiv ─────────── Anime-style artwork
    ├─ DeviantArt ─────── Variety of character designs
    └─ Screenshots ───── Game or anime characters

    Benefits:
    ├─ Maintains design consistency
    ├─ Ensures proper color matching
    └─ Guides facial and clothing details

─────────────────────────────────────────────────────────

1.2 VROID STUDIO
    Free 3D character creator for VTuber models.

    Download:
    ├─ Official Site
    └─ Steam (for easy updates)

    Features:
    ├─ Facial and body customization sliders
    ├─ Hair, clothing, and accessory editors
    └─ Import additional assets from Booth.pm

    Export:
    ├─ .vroid ─── Editable project file
    └─ .vrm ───── For motion capture software

─────────────────────────────────────────────────────────

1.3 ANIMATION SOFTWARE

    ★ VMagicMirror (RECOMMENDED)
      Download VMagicMirror

      ├─ Actively maintained
      ├─ Webcam optional (works without camera)
      ├─ Custom gestures, expressions, poses
      ├─ Lightweight and customizable
      └─ Supports keyboard, mouse, gamepad

    ✗ VSeeFace (NOT RECOMMENDED)
      VSeeFace Website

      └─ No longer maintained - may have compatibility issues

2. Live2D Method (2D Rigged)

⚠ NOTE: I have not personally used this method. Information provided
        is based on research and community recommendations.

Live2D creates animated 2D avatars with layered artwork that moves
naturally. Popular among professional VTubers for its expressive quality.

─────────────────────────────────────────────────────────

2.1 LIVE2D CUBISM
    Software for creating rigged 2D models from layered artwork.

    Live2D Cubism Website

    Pricing:
    ├─ FREE ──────── Limited features, watermark on export
    ├─ PRO ───────── $45/month or $235/year
    └─ Business ──── For commercial use

    Requirements:
    ├─ Layered PSD artwork (separated parts: eyes, mouth, hair, etc.)
    ├─ Artistic skill or commissioned artwork
    └─ Time investment to learn rigging

    Workflow:
    ├─ 1. Create layered artwork in drawing software
    ├─ 2. Import PSD into Live2D Cubism
    ├─ 3. Rig mesh deformations and parameters
    └─ 4. Export model for use in tracking software

─────────────────────────────────────────────────────────

2.2 VTUBE STUDIO
    Motion tracking software for Live2D models.

    VTube Studio Website
    Also available on Steam (free with paid Pro upgrade)

    Features:
    ├─ Webcam or iPhone face tracking
    ├─ Hand tracking support
    ├─ Extensive expression and hotkey system
    └─ Large community of free models to try

    Pricing:
    ├─ FREE ──────── Watermark on screen
    └─ PRO ───────── $25 one-time (removes watermark)

3. Low-End Alternative (2D Static)

For computers that struggle with 3D modeling or real-time tracking.

3.1 VEADOTUBE MINI
    Download Veadotube Mini

    ├─ 2D PNGTuber-style avatars
    ├─ Minimal CPU/GPU usage
    └─ Supports webcam face tracking
